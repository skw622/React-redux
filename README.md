# Stringle

## Guestbook App

The Leading Guestbook App

### Build with

- React
- Material-UI
- Redux
- DraftJS
- ChartJS
- PrismJS
- React Markdown
- React Full Calendar

### Quick start

- Install dependencies: `npm install` or `yarn`
- Start the server: `npm run start` or `yarn start`
- Views are on: `localhost:3000`
- Latest node & react versions recommended
